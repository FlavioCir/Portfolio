Epicode Project - Netflix clone

The 11th Epicode project of the week was about creating a Netflix clone, with a fully functional login and registration form. To ensure the security of users' accounts, a password interceptor was used. In addition, a guard has been included that blocks access to the homepage for unauthenticated users. Finally, it was possible to make the 'likes' related to the user, even after the logout.
